## 📌 Overview
A simple and interactive finance dashboard built using React.
It helps users track income, expenses, and financial insights.

## 🚀 Features
- Add, edit, delete transactions
- View balance, income, expenses
- Category-wise pie chart
- Time-based line chart
- Role-based UI (Admin / Viewer)
- Dark mode support
- Basic insights (top spending category, total transactions)

## 🛠 Tech Stack
- React.js
- JavaScript
- Recharts
- CSS

## 👤 Roles
- Admin → Can add/edit/delete transactions  
- Viewer → Only view data  

## 📊 Insights
- Highest spending category  
- Total transactions  

## ▶️ How to Run
npm install  
npm run dev
